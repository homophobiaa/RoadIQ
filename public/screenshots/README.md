# Снимки на листовки

Сложи тук скрийншоти на сгрешени/трудни листовки.

- Поддържани формати: `.png`, `.jpg`, `.jpeg`, `.webp`
- Един екран = един въпрос (пълен резултатен екран от Avtoizpit/листовки).
- Не е нужно да пишеш JSON — приложението разчита снимките автоматично в браузъра.

След като добавиш файлове, презареди страницата и натисни **„Зареди въпросите“**.
